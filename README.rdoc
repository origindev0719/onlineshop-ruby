HỆ THỐNG BÁN HÀNG ONLINE

Hệ thống bán hàng online mô hình Peer to Peer, ở đó mỗi người đều có thể là người bán và là người mua.


* Ruby version: ruby 2.2.1p85

* Rails version: Rails 4.2.4

* Database: sqlite3

* Others: Bootstrap, paperclip, devise
